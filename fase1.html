Fase 1 — Entrenamiento Baseline

Entrenamiento Base

Cómo se entrena un modelo YOLOv26-cls desde cero para clasificar imágenes médicas. Desde la arquitectura CNN hasta el ajuste fino con AdamW y Cosine Annealing.

1. Arquitectura YOLOv26-cls

YOLOv26-cls no es un detector de objetos; es la rama de clasificación extraída de la familia YOLOv26. La arquitectura se compone de tres bloques funcionales: Backbone, Neck y Head.

🏗️ Backbone

Serie de capas convolucionales que transforman la imagen de entrada en un mapa de características de alto nivel.

  • Conv + BatchNorm + SiLU
  • Bloques C3 (Bottleneck)
  • SPPF (Spatial Pyramid Pooling Fast)

🎯 Neck

En YOLOv26-cls el neck es mínimo. Aplica Global Average Pooling (GAP) que aplana el mapa espacial, haciendo la red invariante a la posición de la lesión.

🧠 Head

Capa lineal (fully connected) que proyecta el vector de 256 características a 2 clases: melanoma y no melanoma.

Nota: C3 es un bloque residual con convoluciones 1×1 y 3×3. SPPF concatena el mismo mapa con max-poolings de distinto tamaño para capturar contexto multi-escala. No hay softmax en la salida: la función de pérdida CrossEntropyLoss aplica softmax internamente por estabilidad numérica.

2. Forward Pass: el viaje de una imagen

Tomemos una imagen de entrada x ∈ R3×224×224. Así es como la red la transforma paso a paso hasta emitir una decisión.

Activación Melanoma No melanoma
Operación de una capa convolucional
y = SiLU ( BN ( W * x + b ) )

Donde W es el kernel aprendible, * es la convolución 2D, BN normaliza por canal y SiLU es la activación Swish.

3. Pérdida: Cross-Entropy ponderada

En datos médicos las clases suelen estar desbalanceadas. En el dataset binario de ejemplo hay aproximadamente 89% no-melanoma y 11% melanoma. Si entrenáramos con Cross-Entropy estándar, el modelo aprendería a decir siempre "no-melanoma" y fallaría en detectar el cáncer.

⚖️ Solución: pesos de clase

Se asigna un peso mayor a la clase minoritaria (melanoma) para forzar al modelo a prestarle atención. Con un factor de suavizado α = 0.3, el peso de melanoma es aproximadamente 2× el de no-melanoma.

🔥 ¿Por qué no pesar 8×?

Sin suavizado, el peso sería ~8× y generaría gradientes explosivos, haciendo el modelo inestable. El factor 0.3 corrige el sesgo sin destruir la convergencia.

Cross-Entropy ponderada
L = − ∑ w yᵢ · log ( ezyᵢ ∑ ezj )

4. Optimización: AdamW + Cosine Annealing

⚡ AdamW

Variante de Adam donde el weight decay (regularización L2) se aplica directamente a los pesos, no al gradiente. Adapta la tasa de aprendizaje por parámetro.

📉 Cosine Annealing

El learning rate comienza alto para explorar rápido, baja suavemente siguiendo una curva coseno, y termina muy bajo para ajuste fino. Evita oscilaciones cerca del óptimo.

Learning rate con Cosine Annealing
η t = η min + 1 2 ( η max − η min ) ( 1 + cos ( t Tmax π ) )

5. Augmentations: distorsión controlada

Las augmentations no son decorativas. En dermatología, el modelo nunca debe memorizar el fondo o la iluminación específica. El objetivo es aprender características invariantes: bordes irregulares, colores, texturas.

Geométricas

fliplr, flipud, rotación 45°

Traslación 10%, escalado 20%. Simulan diferentes posiciones y zoom de la cámara.

Fotométricas

HSV + brillo + contraste

hue=0.02, saturación=0.30, valor=0.25, brillo=0.35, contraste=0.3.

Erasing

RandomErasing (p=0.3)

Oculta regiones aleatorias para forzar al modelo a no depender de una sola zona.

Evitadas

Mosaic y Perspective

Mosaic mezcla 4 imágenes (imposible biológicamente). Perspective distorsiona la piel de forma no fisiológica.

6. Evaluación: índice de Youden

En clasificación binaria desbalanceada, usar threshold = 0.5 es subóptimo. Usamos el índice de Youden para encontrar el punto que maximiza la diferencia entre sensibilidad y falsos positivos.

Índice de Youden
J = Sensibilidad + Especificidad − 1 = TPR − FPR
Protocolo anti data leakage: el threshold óptimo θ* se calcula solo en el conjunto de validación. Luego se aplica fijo a train y test. Esto garantiza que el test mida generalización real, no sobreajuste al threshold.
